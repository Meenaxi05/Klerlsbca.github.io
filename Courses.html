SPECIALIZATION



Cloud Computing

This Specialization dwells on the concepts, methods, and technologies that make up Cloud Computing. It aims to provide and intensive grounding in technologies related to cloud architecture, platform deployment, performance and security. In addition to these technologies, students are exposed to emerging areas of cloud Computing.



OPPORTUNITIES

  • Data Center Technician
  • Cloud Security Specialist
  • Remote Desktop Engineer
  • Cloud Engineer
  • Cloud Architect


Big Data and Data Analytics

Big Data Analytics is probably the fastest evolving issue in the IT world now . Therefore this specialization aims at providing not only the core technologies such as data warehousing, data mining and data modeling, but also gives intensive inputs in areas of machine learning, artificial intelligence and big data analytics.



OPPORTUNITIES

  • Data Engineer
  • Citizen Data Scientist
  • Enterprise Data Analyst
  • Machine learning Engineer
